# Wiki Schema

## Структура vault

```
Obsidian Vault/
├── AGENTS.md          # Эта схема — правила для LLM
├── index.md           # Каталог всех страниц
├── log.md             # Хронология изменений
├── raw/               # Сырые источники (только чтение)
│   └── assets/        # Скачанные изображения
└── wiki/              # Вики-страницы
    ├── concepts/      # Понятия и идеи
    ├── entities/      # Сущности, люди, проекты
    ├── sources/       # Саммари источников
    └── queries/       # Результаты запросов

## Конвенции

- **Ссылки**: используем Obsidian wikilinks `[[Название]]`
- **Frontmatter**: YAML frontmatter с тегами и датами
- **Язык**: русский (пользователь русскоязычный)
- **При ingest**: прочитать сырс → обсудить → создать summary → обновить index → обновить связанные страницы → записать в log

## Операции

- `/ingest [файл/URL]` — добавить новый источник
- `/query [вопрос]` — поиск по wiki и синтез ответа
- `/lint` — проверка здоровья wiki
