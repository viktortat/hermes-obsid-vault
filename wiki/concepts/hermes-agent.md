# Hermes Agent

**Hermes Agent** — opensource приложение от [[Nous Research]] для запуска автономных AI-агентов с самообучением.

## Возможности

- Набор необходимых инструментов (tools) для агента
- Доступ к огромной базе навыков (skills)
- Интеграция с Telegram и другими мессенджерами
- Работа на VPS (24/7)
- Поддержка различных LLM-провайдеров
