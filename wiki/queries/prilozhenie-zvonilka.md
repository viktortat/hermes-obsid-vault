# Приложение-звонилка

Идея: создать приложение, которое совершает телефонные звонки для уведомлений/напоминаний.

## Варианты реализации

- [[Twilio]] API — голосовые звонки через TTS (~3.5–4.5₴/звонок на мобильный UA)
- [[Vonage]] — альтернатива, дешевле (~2.9₴/sms)
- [[Vodafone Україна]] — напрямую через оператора (немає публічного API)
- Telegram-уведомления — бесплатно, но нужен интернет

## Использование с [[Hermes Agent]]

- Интеграция через cron job для напоминаний
- Звуковое оповещение (звонок) как более надёжный способ, чем Telegram
