# Дрессировка и воспитание личного автономного AI‑агента на VPS (Hermes Agent вместо OpenClaw)

Подробная статья о настройке [[Hermes Agent]] от Nous Research на VPS.

## Основные темы

- **Архитектура Hermes Agent** — разбор компонентов, tools, навыков (skills)
- **Установка** — пошаговая инструкция для VPS (для «чайников»)
- **Воспитание агента** — 4 реальных кейса использования:
  1. Мониторинг серверов
  2. Автоматизация задач в Telegram
  3. Работа с базами знаний
  4. Интеграция с внешними сервисами
- **Выбор LLM** — особенности различных моделей в качестве «мозгов» агента
- **Плюсы, минусы и ограничения** Hermes Agent

## Ключевые инсайты

- Hermes Agent — opensource, свободен от корпоративных инфраструктур
- Поддержка Telegram и других мессенджеров
- Самообучение и огромная база навыков (skills)
- Требует «дрессировки» — агента нужно настраивать под свои задачи
